# Memulai dengan ERP ini

Selamat datang di aplikasi. Manual singkat ini ada agar `php artisan help:reindex` punya konten untuk diindeks.

## Masuk (login)

Gunakan kredensial Anda di halaman **Login**. Jika lupa kata sandi, gunakan **Change Password** dari menu pengguna setelah administrator mereset akun Anda.

## Menemukan halaman lebih cepat (pencarian menu)

Di layar lebar, gunakan kolom **Search Menu here** di bilah atas untuk menyaring halaman yang Anda boleh buka (izin sama seperti sidebar). Gunakan panah untuk berpindah di daftar, **Enter** untuk membuka, **Escape** untuk menutup. **Ctrl+K** (Windows/Linux) atau **Cmd+K** (macOS) memfokuskan kolom pencarian jika Anda tidak sedang mengetik di kolom lain.

## Di mana menemukan bantuan

Klik ikon tanda tanya (**?**) di bilah atas untuk membuka panel **Help**. Ajukan pertanyaan cara pakai di tab **How-to** atau kirim laporan bug dan permintaan fitur di tab **Report / request**.

Contoh topik di HELP: **rekonsiliasi bank**, **RAB / Anggaran**, dan **scan nota BBM untuk realisasi** (tombol **Scan Fuel Receipts** di halaman detail realisasi — hanya nota BBM).

## Setelah dokumentasi berubah

Administrator menjalankan `php artisan help:reindex` di server agar jawaban HELP mengikuti manual terbaru.
